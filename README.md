# e-Portfolio React

Full-stack developer portfolio showcasing my projects and skills

This is a React-based e-portfolio application designed to showcase personal projects, skills, and contact information. The application is structured to provide a seamless user experience with a modern design.

## Features

- **Home Page**: A welcoming landing page that introduces the portfolio owner.
- **About Page**: Information about the portfolio owner, including background and skills.
- **Projects Page**: A showcase of completed projects with descriptions and links.
- **Contact Page**: A form for visitors to reach out to the portfolio owner.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons

## Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/michaelpo1738/e-portfolio.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd e-portfolio-react
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm run dev
   ```

5. **Open your browser** and go to `http://localhost:5173` to see the application in action.

## Folder Structure

- `src/`: Contains all the source code for the application.
  - `components/`: Reusable components like Header, Footer, and Navbar.
  - `pages/`: Individual pages of the application.
  - `hooks/`: Custom hooks for managing state.
  - `context/`: Context API for theme management.
  - `services/`: Functions for handling email sending.
  - `data/`: Mock data for projects and profile.
  - `styles/`: CSS files for styling the application.
  - `types/`: TypeScript types and interfaces.
  - `__tests__/`: Unit tests for components.
